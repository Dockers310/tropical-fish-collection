## English

A client-side Fabric mod that helps you track and collect **all 3072 variants of tropical fish** in Minecraft.

- 🐠 Automatically records every fish bucket that enters your inventory  
- 📊 Collection screen with a grid, search, sorting, progress bar, and tooltips  
- 🟢🟡🔴 **Bucket indicators** (green = collected, yellow = duplicate, red = new)  
- 💡 **Highlight uncollected fish in the world** (optional glow effect)  
- 🔔 Smart grouped toasts – a single notification for a batch of new fish  
- 🏆 Local leaderboard (your stats only; multiplayer leaderboard requires the mod on the server)  
- 🎯 Milestone toasts every 5% of completion  
- ⚙️ Extensive settings via Mod Menu (YACL)

### Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.1  
2. Download [Fabric API](https://modrinth.com/mod/fabric-api), [Mod Menu](https://modrinth.com/mod/modmenu) and [YetAnotherConfigLib](https://modrinth.com/mod/yacl)  
3. Place all `.jar` files into your `mods` folder  
4. Done! Press **K** to open the collection screen

### Building from source

```bash
git clone https://github.com/yourname/tropical-fish-collection
cd tropical-fish-collection
./gradlew build
# The mod will be in build/libs/
```

## Русский
# Tropical Fish Collection

Клиентский мод для Fabric, позволяющий отслеживать и собирать **все 3072 варианта тропических рыб** в Minecraft.

- 🐠 Автоматически запоминает каждое ведро с рыбой, попавшее в инвентарь
- 📊 Экран коллекции с сеткой, поиском, сортировками, прогресс‑баром и подсказками
- 🟢🟡🔴 **Цветовые индикаторы на вёдрах** (зелёный – собрана, жёлтый – дубликат, красный – новая)
- 💡 **Подсветка несобранных рыб в мире** (опциональное свечение, настраиваемый цвет и дальность)
- 🔔 Умные групповые уведомления — одно сообщение на пачку новых рыб
- 🏆 Локальная таблица лидеров (только ваша статистика; многопользовательский топ требует мод на сервере)
- 🎯 Поздравительные тосты за каждые 5% заполнения коллекции
- ⚙️ Обширные настройки через Mod Menu (YACL)

### Установка

1. Установите [Fabric Loader](https://fabricmc.net/use/) для Minecraft 1.21.1
2. Скачайте [Fabric API](https://modrinth.com/mod/fabric-api), [Mod Menu](https://modrinth.com/mod/modmenu) и [YetAnotherConfigLib](https://modrinth.com/mod/yacl)
3. Поместите все `.jar` файлы в папку `mods`
4. Готово! Нажмите **K**, чтобы открыть экран коллекции

### Сборка из исходников

```bash
git clone https://github.com/yourname/tropical-fish-collection
cd tropical-fish-collection
./gradlew build
# Готовый мод появится в папке build/libs/
```
